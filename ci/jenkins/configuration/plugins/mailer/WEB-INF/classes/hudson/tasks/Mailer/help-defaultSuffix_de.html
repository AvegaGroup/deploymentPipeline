Falls die E-Mail-Adressen der Benutzer systematisch durch Anhängen eines Suffixes gebildet werden können, geben Sie hier den Suffix an. Andernfalls lassen Sie das Feld frei. Beachten Sie, das Benutzer ihre E-Mail-Adressen trotzdem selektiv überschreiben können.

Beispiel: Steht in diesem Feld @acme.org, wird dem Benutzer foo automatisch die E-Mail-Adresse foo@acme.org zugeordnet.
