Delegiert die Authentifizierung an das Benutzerverzeichnis des darunterliegenden Unix-Betriebssystems. Mit dieser Einstellung melden sich Benutzer an Jenkins mit ihren Unix-Benutzernamen und -Passwörtern an.

Dieser Modus erlaubt es darüberhinaus, Unix Benutzergruppen zur Autorisierung einzusetzen. Dies erlaubt Regeln wie beispielsweise "Alle in der Gruppe 'developers' sollen administrativen Zugriff erhalten".

Dieser Modus ist über die PAM-Bibliothek, realisiert, die über einen eigenständigen Konfigurationsmechanismus verfügt. Sie arbeitet mit Erweiterungen wie NIS zusammen.
